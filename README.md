# Official Website of Stamped (worldmap)

Static marketing and legal site for **Stamped**, a travel tracker that turns the
countries and regions you visit into a collectible world map.

## Pages

| File | Purpose |
| --- | --- |
| `index.html` | Landing page |
| `privacy.html` | Privacy Policy (App Store / Google Play) |
| `terms.html` | Terms of Service |
| `support.html` | Support & FAQ |
| `styles.css` | Shared styles |

## Hosting (GitHub Pages)

1. Repository **Settings → Pages**
2. **Source:** Deploy from a branch → `main` / root (`/`)
3. Once published, the pages are available at:
   - `https://smri2170.github.io/Official-Website-of-worldmap/`
   - Privacy: `https://smri2170.github.io/Official-Website-of-worldmap/privacy.html`
   - Terms: `https://smri2170.github.io/Official-Website-of-worldmap/terms.html`
   - Support: `https://smri2170.github.io/Official-Website-of-worldmap/support.html`

Use the Privacy and Support URLs in App Store Connect and Google Play Console.
